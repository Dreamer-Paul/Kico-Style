# Kico-Style

一个可口的极简响应式前端框架。

## 使用方法

1. `Star` 本项目
2. 从这里 [下载](https://github.com/Dreamer-Paul/Kico-Style/archive/master.zip) 框架源码
3. 在你的项目中引用相关文件，参考 [文档](https://works.paugram.com/style) 快速使用
